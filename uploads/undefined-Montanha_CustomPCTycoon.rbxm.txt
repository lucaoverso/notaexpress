
-- Este é um modelo básico de montanha no estilo low poly inspirado em Custom PC Tycoon.
-- O arquivo precisa ser convertido para um verdadeiro .rbxm no Roblox Studio.
-- Instruções:
-- 1. Crie Parts com base marrom (terra) e topo verde (grama), empilhadas com formas suaves.
-- 2. Use Material 'SmoothPlastic' e cores RGB próximas de:
--    - Marrom: RGB(101, 67, 33)
--    - Verde: RGB(53, 153, 51)
-- 3. Agrupe os blocos e salve como modelo (Model > Save to File) para gerar um .rbxm verdadeiro.

-- Este é apenas um guia. Para montar diretamente no Roblox Studio, siga os passos acima.
